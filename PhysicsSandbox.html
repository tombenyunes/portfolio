Physics Sandbox

Project Description

A simple physics engine. Features a customisable spring-mass-damper model, calculates gravity/friction, soft and hard collisions, and interpolates movement.
All forces are calculated relative to masses.

Different object types can be placed, selected and moved individually, and a variety of properties can be customized.

Technical Info

  • C++
  • openFrameworks
